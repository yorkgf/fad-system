# 学生处老师操作指南

## 角色概述

学生处老师拥有校级管理权限，可执行 FAD、查看全校/班级统计、管理寝室清洁与停课，以及查询导出所有数据。但不可发放 FAD/Reward、不可管理教学票和寝室表扬兑换。

---

## 首页 — 功能导航

登录后进入首页，你可以看到以下功能卡片：

| 卡片名称 | 说明 |
|----------|------|
| **录入记录** | 添加学生行为记录 |
| **我的记录** | 查看已录入的记录 |
| **FAD执行** | 管理FAD执行状态 |
| **学校FAD统计** | 查看全校FAD数据统计 |
| **班级FAD统计** | 查看本班FAD数据统计（需为班主任） |
| **寝室清扫** | 记录寝室清扫情况 |
| **最佳寝室排名** | 查看优秀寝室排行 |
| **网课违规使用电子产品** | 暂停网课资格名单 |
| **未交手机名单** | 查看未交手机学生 |
| **约谈/停课管理** | 管理约谈和停课记录 |
| **数据查询** | 查询所有数据 |
| **日程管理** | 预约教师面谈时间 |

**你看不到的卡片：** FAD通知单发放、寝室表扬兑奖、教学票兑奖（仅系统管理员可见）

页面右上角点击用户名可展开菜单：**教师信息** / **修改密码** / **退出登录**。语言切换按钮可切换中英文界面。

## 一、录入记录

> 首页点击 **「录入记录」** 卡片进入

### 1.1 可用记录类型

你可以插入以下常规记录类型：

- 早点名迟到、寝室迟出、未按规定返校、擅自进入会议室或接待室
- 寝室表扬、寝室批评、寝室垃圾未倒
- 21:30后交还手机(22:00前)、22:00后交还手机
- Teaching FAD Ticket、Teaching Reward Ticket

**不可插入：** FAD、Reward、上网课违规使用电子产品（系统管理员专属）

### 1.2 操作步骤

1. **选择记录类型**：在「记录类型」下拉框中选择
2. **选择学期**：在「学期」下拉框中选择（默认当前学期）
3. **选择学生**：
   - **寝室类记录**（寝室迟出、寝室表扬、寝室批评、寝室垃圾未倒）：先在「寝室」下拉框选择寝室号 → 在「寝室学生」下拉框勾选学生
   - **其他记录**：在「学生」搜索框输入姓名搜索，从结果中勾选（支持多选）
4. **填写日期**：在「记录日期」选择器中选择（默认今天）
5. **填写事由**：「记录事由」文本框（Teaching Ticket 类型必填）
6. 点击 **「提交」** 按钮

已选学生显示为标签，点击 × 可移除。点击 **「重置」** 清空整个表单。

### 1.3 特殊字段

| 记录类型 | 额外字段 | 说明 |
|----------|----------|------|
| Teaching FAD/Reward Ticket | **票据数量**（下拉框） | 可选 1~6 张 |

### 1.4 累计规则

提交成功后系统自动检查累计阈值，达到时弹出提示：

| 记录类型 | 阈值 | 结果 |
|----------|------|------|
| 早点名迟到 | 2 次 | 自动生成 FAD |
| 寝室迟出 | 2 次 | 自动生成 FAD |
| 未按规定返校 | 2 次 | 自动生成 FAD |
| 擅自进入会议室或接待室 | 2 次 | 自动生成 FAD |
| 寝室批评 | 5 次 | 自动生成 FAD |
| 寝室垃圾未倒 | 2 次 | 自动生成寝室批评 |
| Teaching FAD Ticket | 3 张 | 自动生成 FAD |
| 22:00后交还手机 | 每次 | 立即生成 FAD |
| 寝室表扬 | 10 次 | 提示可兑换 Reward |
| Teaching Reward Ticket | 6 张 | 提示可兑换 Reward |

> **注意：** 寝室表扬/批评/垃圾未倒不允许同一学生同一天重复录入。

## 二、我的记录

> 首页点击 **「我的记录」** 卡片进入

### 2.1 筛选栏

| 筛选项 | 说明 |
|--------|------|
| **记录类型** | 下拉框，选择后自动刷新 |
| **FAD来源类型** | 仅查看 FAD 记录时显示 |
| **学期** | 下拉框 |
| **教师** | 下拉框（可搜索）— 学生处老师可按教师筛选 |
| **学生姓名** | 输入框 |
| **班级** | 下拉框（可搜索） |
| **记录日期范围** | 日期范围选择器 |
| **导出CSV** | 绿色按钮，导出当前筛选结果 |

### 2.2 状态标签含义

| 记录类型 | 标签 | 颜色 | 含义 |
|----------|------|------|------|
| FAD | 已冲销 | 绿色 | 已被 Reward 冲销 |
| FAD | 已执行未冲销 | 橙色 | 已执行但未冲销 |
| FAD | 未执行 | 红色 | 尚未执行 |
| 寝室批评 | 已累计FAD / 未累计FAD | 红/蓝 | 是否已触发 FAD |
| 寝室表扬 | 已累计Reward / 未累计Reward | 红/蓝 | 是否已兑换 |
| 寝室垃圾未倒 | 已累计批评 / 未累计批评 | 红/蓝 | 是否已触发寝室批评 |
| 其他 | 有效 | 蓝色 | 记录正常 |

### 2.3 撤回记录

点击记录右侧 **「撤回」** 按钮：
1. 弹出撤回确认对话框，顶部橙色警告说明影响
2. 如有级联记录（由该记录累计生成的 FAD/寝室批评），会列出将被一并删除的记录
3. 在「撤回原因」文本框中输入原因（必填）
4. 点击红色 **「确认撤回」** 按钮

**不可撤回的情况**（按钮灰色不可点击，鼠标悬停显示原因）：
- Reward 记录
- 已发放的 FAD 或其他记录
- 已执行/已冲销的 FAD
- 已兑换 Reward 的寝室表扬/教学票
- 非自己录入的记录（系统管理员录入的除外，学生处老师可撤回所有人的记录）

### 2.4 分页

底部分页控件，可选每页 10 / 20 / 50 / 100 条。

## 三、FAD 执行

> 首页点击 **「FAD执行」** 卡片进入

列出所有**未执行**的 FAD 记录。

### 3.1 筛选

- **选择学期** — 下拉框，选后自动刷新
- **FAD来源类型** — 下拉框（可清除）

### 3.2 操作

**单条执行：** 点击右侧 **「执行」** 按钮 → 确认对话框 → 点击 **「确认执行」**

**批量执行：** 勾选多条记录 → 点击 **「批量执行 (N条)」** → 确认

FAD 来源类型标签颜色：寝室类=橙色、教学类=红色、其他=蓝色。

---

## 四、学校FAD统计

> 首页点击 **「学校FAD统计」** 卡片进入

### 4.1 筛选

| 筛选项 | 说明 |
|--------|------|
| **统计周期** | 春季 / 秋季 / 学年 |
| **班级** | 下拉框（可搜索），可选特定班级或全校 |
| **来源** | 全部 / 寝室类 / 教学类 / 电子产品 / 其他 |

### 4.2 数据概览

页面顶部四个统计卡片：**总FAD数**、**已执行**（绿色）、**已发放**（蓝色）、**已冲销**（灰色）

### 4.3 来源类型分布表

展示各来源类型的 FAD 数量分布。

### 4.4 未执行学生名单

选择了特定班级时显示，列出该班有未执行 FAD 的学生。

### 4.5 学生FAD排名

FAD 数量排名靠前的学生，点击 **▶ 展开箭头** 查看明细记录（日期、事由、老师、来源类型、执行/冲销状态）。

来源分布标签：寝室=橙色、教学=红色、电子产品=蓝色、其他=灰色。

## 五、班级FAD统计

> 首页点击 **「班级FAD统计」** 卡片进入（需为班主任）

功能与学校FAD统计相同，但自动锁定为你所带班级。

## 六、寝室清扫

> 首页点击 **「寝室清扫」** 卡片进入

### 6.1 页面说明

列表显示有「寝室垃圾未倒」记录的学生。标签颜色：≥3 次=红色，<3 次=橙色。

### 6.2 操作

点击 **「确认清理」** 按钮（次数 < 3 时灰色不可点击）→ 确认对话框 → 点击 **「确认」**

确认后系统将未清扫记录转换为寝室批评。

---

## 七、网课违规使用电子产品

> 首页点击 **「网课违规使用电子产品」** 卡片进入

只读列表，显示被暂停网课资格的学生。状态列自动判断：**禁止上课**（红色）或 **已恢复**（绿色）。

---

## 八、未交手机名单

> 首页点击 **「未交手机名单」** 卡片进入

显示**当天**的手机交还记录。迟交类型标签：21:30后=橙色，22:00后=红色。是否触发FAD列：22:00后="是"（红色），21:30后="否"（灰色）。

点击 **「刷新」** 按钮更新数据。仅显示当天记录。

---

## 九、数据查询

> 首页点击 **「数据查询」** 卡片进入

### 9.1 筛选条件

| 筛选项 | 说明 |
|--------|------|
| **记录类型** | 12 种可选 |
| **学期** | 可清除 |
| **班级** | 可搜索、可清除 |
| **学生** | 输入框，按回车搜索 |
| **日期范围** | 日期范围选择器 |
| **来源类型** | 仅 FAD 记录时显示 |

操作按钮：**「查询」**（蓝色）/ **「重置」** / **「导出」**（绿色）

### 9.2 结果展示

不同记录类型显示不同列和状态标签。FAD 记录额外显示来源类型和状态（已冲销/已执行未冲销/未执行）。底部显示总记录数，分页可选 20/50/100/200 条。

## 十、约谈/停课管理

> 首页点击 **「约谈/停课管理」** 卡片进入

### 10.1 FAD 处罚规则

页面顶部橙色提示框：
- **3 次** FAD → 约谈家长
- **6 次** FAD → 停课处理
- **9 次** FAD → 劝退处理

### 10.2 四个标签页

#### 约谈名单（FAD 3~5 次）

- 点击 **▶ 展开** 查看 FAD 明细
- 点击 **「已约谈」**（蓝色按钮）→ 确认后标记为已约谈
- 点击 **打印图标** → 下载该学生 FAD 记录 PDF

#### 停课名单（FAD 6~8 次）

- 点击 **「已停课」**（红色按钮）→ 弹出停课处理对话框：
  - 选择**停课开始日期**
  - 输入**停课天数**（1~30 天）
  - 点击 **「确认停课」**

#### 劝退名单（FAD ≥ 9 次）

- 点击 **「已劝退」**（红色按钮）→ 确认（**不可撤销**）

#### 历史记录

所有约谈/停课记录列表，类型标签：约谈=绿色、停课=红色。

## 十一、日程管理

> 首页点击 **「日程管理」** 卡片进入

### 11.1 创建时段

1. **选择日期**：日期选择器选择多个日期
2. 勾选 **上午时段** 和/或 **下午时段**，设置开始和结束时间
3. **时间分割**：选择每段时长 10 / 15 / 20 / 30 分钟
4. **地点**（必填）：选择"线下"或"腾讯会议"
5. **备注**：可选
6. 点击 **「创建时段」** 按钮

> 如果新时段与已有时段冲突，系统会弹出红色错误提示并列出冲突详情。

### 11.2 管理时段

- 未预约的时段可点击红色 **「删除」** 按钮删除
- 已有预约的时段显示 **「已有预约」** 标签，不可删除
- 点击 **「删除全部未预约」** 可批量删除（需确认，**不可恢复**）
- 点击 **「我的日程」** 进入日历视图

---

## 十二、设置

### 修改密码

> 右上角用户名 → **「修改密码」**

1. 输入「当前密码」
2. 输入「新密码」（≥ 6 位）
3. 输入「确认新密码」（须一致）
4. 点击 **「确认修改」**

修改成功后自动退出登录，需用新密码重新登录。

### 教师信息

> 右上角用户名 → **「教师信息」**

查看和编辑个人信息。

---

## 重要注意事项

1. **你不可以发放 FAD/Reward**，需联系系统管理员操作
2. **你不可以修改 FAD 来源类型**，仅系统管理员可操作
3. 已发放的记录不可撤回
4. 撤回记录会级联删除由该记录累计生成的 FAD/寝室批评
5. 寝室表扬兑换 Reward 和教学票兑奖需联系系统管理员
6. 寝室表扬/批评/垃圾未倒不允许同一学生同一天重复录入
