# 保洁阿姨操作指南

## 角色概述

保洁阿姨拥有系统最基础的权限，仅可插入学生记录、查看自己的记录和修改密码。不可访问 FAD 管理、统计、寝室管理、日程管理等任何管理功能。

---

## 首页 — 功能导航

登录后进入首页，你可以看到以下功能卡片：

| 卡片名称 | 说明 |
|----------|------|
| **录入记录** | 添加学生行为记录 |
| **我的记录** | 查看已录入的记录 |

**你看不到的卡片：** FAD执行、FAD发放、所有统计页面、寝室管理、电子产品违规、手机名单、约谈/停课管理、教学票兑奖、数据查询、日程管理

页面右上角点击用户名可展开菜单：
- **修改密码** — 修改登录密码
- **退出登录** — 退出当前账号

---

## 一、录入记录

> 首页点击 **「录入记录」** 卡片进入

### 1.1 可用记录类型

你可以插入以下记录类型：

| 记录类型 | 说明 |
|----------|------|
| 早点名迟到 | 学生早点名迟到 |
| 寝室迟出 | 学生寝室迟出 |
| 未按规定返校 | 学生未按规定返校 |
| 擅自进入会议室或接待室 | 学生擅自进入会议室或接待室 |
| 寝室表扬 | 寝室正面行为表扬 |
| 寝室批评 | 寝室负面行为批评 |
| 寝室垃圾未倒 | 寝室垃圾未清理 |
| 21:30后交还手机(22:00前) | 手机迟交（不触发FAD） |
| 22:00后交还手机 | 手机迟交（立即触发FAD） |
| Teaching FAD Ticket | 教学纪律票 |
| Teaching Reward Ticket | 教学奖励票 |

**不可插入：** FAD、Reward、上网课违规使用电子产品（管理员专属）

### 1.2 操作步骤

1. **选择记录类型**：在「记录类型」下拉框中选择

2. **选择学期**：在「学期」下拉框中选择（默认为当前学期）

3. **选择学生**：
   - **寝室类记录**（寝室迟出、寝室表扬、寝室批评、寝室垃圾未倒）：
     - 先在「寝室」下拉框中选择寝室号
     - 系统自动加载该寝室的学生
     - 在「寝室学生」下拉框中勾选需要录入的学生（可多选）
   - **其他记录**：
     - 在「学生」搜索框中输入学生姓名
     - 从下拉搜索结果中勾选学生（可多选）

4. **填写日期**：在「记录日期」日期选择器中选择日期（默认为今天）

5. **填写事由**：在「记录事由」文本框中输入原因
   - **必填**的记录类型：Teaching FAD Ticket、Teaching Reward Ticket
   - 其他类型为可选

6. 点击 **「提交」** 按钮提交记录

**补充说明：**
- 已选中的学生会显示为标签，点击标签上的 **×** 可移除该学生
- 点击 **「重置」** 按钮可清空整个表单重新填写

### 1.3 特殊字段

| 记录类型 | 额外字段 | 说明 |
|----------|----------|------|
| Teaching FAD Ticket | **票据数量**（下拉框） | 可选 1~6 张，一次性为同一学生插入多张 |
| Teaching Reward Ticket | **票据数量**（下拉框） | 可选 1~6 张，一次性为同一学生插入多张 |

### 1.4 累计规则与自动触发

提交记录成功后，系统会自动检查该学生同类型记录的累计数量。当达到阈值时，页面会弹出提示框通知你：

| 记录类型 | 累计阈值 | 自动触发结果 |
|----------|----------|-------------|
| 早点名迟到 | 2 次 | 自动生成 FAD 记录 |
| 寝室迟出 | 2 次 | 自动生成 FAD 记录 |
| 未按规定返校 | 2 次 | 自动生成 FAD 记录 |
| 擅自进入会议室或接待室 | 2 次 | 自动生成 FAD 记录 |
| 寝室批评 | 5 次 | 自动生成 FAD 记录 |
| 寝室垃圾未倒 | 2 次 | 自动生成「寝室批评」记录（寝室批评累计到 5 次会再触发 FAD） |
| Teaching FAD Ticket | 3 张 | 自动生成 FAD 记录 |
| 22:00后交还手机 | 每次 | **每次录入都立即生成 FAD** |
| 寝室表扬 | 10 次 | 弹出提示可兑换 Reward（需联系管理员操作） |
| Teaching Reward Ticket | 6 张 | 弹出提示可兑换 Reward（需联系管理员操作） |

> **特别注意：** "21:30后交还手机(22:00前)" **不会**触发 FAD，但 "22:00后交还手机" **每次**都会立即触发 FAD。请注意区分这两种类型。

### 1.5 提交结果说明

- **全部成功**：显示绿色成功消息，如果触发了累计规则还会列出生成的 FAD/警告数量
- **部分失败**：显示橙色提示，说明哪些学生提交失败
  - 常见原因：寝室表扬/批评/垃圾未倒**同一学生同一天不能重复录入**
- **全部失败**：显示红色错误消息

## 二、我的记录

> 首页点击 **「我的记录」** 卡片进入

此页面用于查看你已经录入的所有记录。

### 2.1 筛选栏

页面顶部有一排筛选控件：

| 筛选项 | 类型 | 说明 |
|--------|------|------|
| **记录类型** | 下拉框 | 选择要查看的记录类型，选择后列表自动刷新 |
| **FAD来源类型** | 下拉框 | 仅当查看 FAD 记录时显示 |
| **学期** | 下拉框 | 按学期筛选 |
| **学生姓名** | 输入框 | 输入学生姓名搜索 |
| **班级** | 下拉框（可搜索） | 按班级筛选 |
| **记录日期范围** | 日期范围选择器 | 选择开始和结束日期 |
| **导出CSV** | 绿色按钮 | 将筛选结果导出为 CSV 文件下载 |

> 你只能看到自己录入的记录，无法查看其他教师的记录。

### 2.2 记录列表

列表以表格形式展示每条记录的详细信息：

| 列名 | 说明 |
|------|------|
| 学生 | 学生姓名 |
| 班级 | 所在班级 |
| 记录类型 | 记录的类型 |
| 记录日期 | 格式 YYYY-MM-DD |
| 记录事由 | 鼠标悬停可查看完整内容 |
| 记录老师 | 显示你的姓名 |
| 状态 | 彩色状态标签（见下表） |
| 操作 | 撤回按钮 |

### 2.3 状态标签含义

| 记录类型 | 标签文字 | 标签颜色 | 含义 |
|----------|----------|----------|------|
| FAD | 已冲销 | 绿色 | 已被 Reward 冲销 |
| FAD | 已执行未冲销 | 橙色 | FAD 已执行但未冲销 |
| FAD | 未执行 | 红色 | FAD 尚未执行 |
| 寝室批评 | 已累计FAD | 红色 | 该批评已触发生成 FAD |
| 寝室批评 | 未累计FAD | 蓝色 | 尚未达到累计阈值 |
| 寝室表扬 | 已累计Reward | 红色 | 已参与 Reward 兑换 |
| 寝室表扬 | 未累计Reward | 蓝色 | 尚未兑换 |
| 寝室垃圾未倒 | 已累计批评 | 红色 | 已触发寝室批评 |
| 寝室垃圾未倒 | 未累计批评 | 蓝色 | 未达到阈值 |
| 其他 | 有效 | 蓝色 | 记录正常有效 |

### 2.4 撤回记录

点击记录右侧的 **「撤回」** 按钮：

1. 弹出撤回确认对话框
2. 顶部有橙色警告，说明撤回后的影响
3. 如果该记录累计触发了其他记录（如 FAD 或寝室批评），会显示**级联记录列表**，列出将被一并删除的关联记录
4. 在「撤回原因」文本框中输入撤回原因（**必填**）
5. 点击红色 **「确认撤回」** 按钮执行撤回

**以下情况「撤回」按钮显示为灰色不可点击**（鼠标悬停可看到原因）：

| 情况 | 提示信息 |
|------|----------|
| Reward 记录 | Reward记录不可撤回 |
| 已发放的记录 | 已发放的记录不可撤回 |
| 已发放的 FAD | 已发放的FAD不可撤回 |
| 已执行的 FAD | 已执行的FAD不可撤回 |
| 已冲销的 FAD | 已冲销的FAD不可撤回 |
| 已兑换Reward的寝室表扬 | 已兑换Reward的寝室表扬不可撤回 |
| 已兑换Reward的教学票 | 已兑换Reward的教学票不可撤回 |

### 2.5 分页

列表底部有分页控件，可选择每页显示 **10 / 20 / 50 / 100** 条记录。

## 三、修改密码

> 点击右上角用户名 → 下拉菜单中选择 **「修改密码」**

### 操作步骤

1. 在「当前密码」输入框中输入你现有的密码
2. 在「新密码」输入框中输入新密码（**不少于 6 位**）
3. 在「确认新密码」输入框中再次输入新密码（**必须与上面完全一致**）
4. 点击 **「确认修改」** 按钮

修改成功后，系统会自动退出登录并跳转到登录页面。你需要使用新密码重新登录。

> **注意：** 如果两次输入的新密码不一致，会提示"两次输入的密码不一致"，需要重新输入。

---

## 重要注意事项

1. 你只能查看和撤回**自己录入**的记录，无法查看或操作其他教师的记录
2. 撤回一条记录后，由该记录累计触发生成的 FAD 或寝室批评也会被**同时删除**
3. 寝室表扬/批评/垃圾未倒**同一学生同一天不能重复录入**
4. 寝室表扬累计 10 次可兑换 Reward、Teaching Reward Ticket 累计 6 张可兑换 Reward，但兑换操作需联系管理员处理
5. 如果你需要访问更多功能（如日程管理、统计查询等），请联系系统管理员调整你的权限组别
