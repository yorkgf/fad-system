# 班主任附加功能说明

> 本文档是班主任角色的**补充说明**，适用于所有担任班主任的用户。班主任不是独立的用户组，而是在现有角色基础上的附加角色。你的基本功能权限仍由所在角色决定，请结合对应角色的操作指南一并阅读。

---

## 什么是班主任？

班主任是对某个班级负有管理责任的教师。系统通过数据库中 `All_Classes` 集合的 `HomeTeacherEmail` 字段来识别班主任身份——当你的登录邮箱与某个班级的 `HomeTeacherEmail` 一致时，系统自动将你识别为该班级的班主任。

> **你无需手动设置班主任身份。** 系统在你登录后自动检测。如果你应该是班主任但未被识别，请联系系统管理员检查 `All_Classes` 中对应班级的 `HomeTeacherEmail` 是否与你的登录邮箱一致。

## 班主任专属功能

### 一、班级FAD统计

> 首页点击 **「班级FAD统计」** 卡片进入

这是班主任最核心的专属功能。该卡片**仅在你被识别为班主任时出现**，非班主任用户在首页看不到此卡片。

#### 1.1 页面顶部提示

进入页面后，顶部蓝色信息提示框会显示：

> 正在查看班级：**XX班**

确认你所管理的班级信息正确。

#### 1.2 筛选条件

| 筛选项 | 类型 | 可选值 | 说明 |
|--------|------|--------|------|
| **统计周期** | 下拉框 | 春季(Spring) / 秋季(Fall) / 学年 | 选择后自动刷新数据 |
| **来源** | 下拉框（可清除） | 全部 / 寝室类 / 教学类 / 电子产品 / 其他 | 按 FAD 来源类型筛选 |

> **注意：** 与「学校FAD统计」不同，此页面**没有班级选择框**，因为数据自动锁定为你所带的班级。系统管理员或学生处老师如需查看其他班级数据，请使用「学校FAD统计」功能。

#### 1.3 统计概览卡片

页面显示四个数据卡片，一目了然掌握本班 FAD 情况：

| 卡片 | 数字颜色 | 说明 |
|------|----------|------|
| **本班FAD总数** | 黑色 | 当前筛选条件下的本班 FAD 记录总数 |
| **已执行** | 绿色 | 已被标记为执行的 FAD 数量 |
| **未执行** | 橙色 | 尚未执行的 FAD 数量（需重点关注） |
| **已冲销** | 灰色 | 被 Reward 冲销的 FAD 数量 |

#### 1.4 未执行FAD学生警示列表

当本班有学生存在**未执行的 FAD** 时，页面会显示一个红色标题的警示卡片：

**「未执行FAD学生列表」** 右侧有红色标签显示人数（如 `共3人`）

列表内容：

| 列 | 说明 |
|----|------|
| 学生 | 学生姓名 |
| 未执行数 | 红色标签，显示该学生未执行的 FAD 数量 |

> 这个列表帮助你快速定位需要跟进的学生。

#### 1.5 本班学生FAD明细

下方表格列出本班所有有 FAD 记录的学生（最多显示前 20 名）：

| 列 | 说明 |
|----|------|
| **▶ 展开箭头** | 点击可展开查看该学生的 FAD 明细记录 |
| 学生 | 学生姓名 |
| 班级 | 所在班级 |
| **FAD总数** | 红色标签，可点击列头排序 |
| 已执行 | 已执行的 FAD 数量 |
| **未执行** | 未执行数量，>0 时显示红色标签 |
| 已冲销 | 被 Reward 冲销的数量 |

#### 1.6 展开查看FAD明细

点击某个学生行首的 **▶ 展开箭头**，下方会展开一个子表格，标题为：

> **XXX 的FAD明细记录**

明细记录包含以下列：

| 列 | 说明 |
|----|------|
| 记录日期 | FAD 的记录日期，格式 YYYY/MM/DD |
| 记录事由 | FAD 的原因说明 |
| 记录老师 | 录入该 FAD 的教师 |
| 来源类型 | 彩色标签，表示 FAD 的来源 |
| 执行状态 | 绿色 **已执行** 或 红色 **未执行** |
| 冲销状态 | 灰色 **已冲销** 或 橙色 **未冲销** |

**来源类型标签颜色对照：**

| 来源 | 标签颜色 | 含义 |
|------|----------|------|
| 寝室类 | 橙色 | 来自寝室相关记录累计（寝室迟出、寝室批评、22:00后交手机等） |
| 教学类 | 红色 | 来自 Teaching FAD Ticket 累计或上网课违规 |
| 电子产品 | 蓝色 | 来自上网课违规使用电子产品 |
| 其他 | 灰色 | 来自早点名迟到、擅自进入会议室等累计 |

### 二、邮件通知

当其他教师为你班上的学生录入新记录时，系统会**自动发送邮件通知**到你的班主任邮箱。

通知内容包含：
- 被记录的学生姓名和班级
- 记录类型
- 记录日期和事由
- 录入教师

> **以下记录类型不会发送邮件通知：** 寝室表扬、Teaching Reward Ticket、Reward。这些正面记录不触发邮件。

邮件通知是自动的，无需你做任何配置。如果你没有收到邮件通知，请联系系统管理员检查：
1. 你的邮箱是否正确配置在 `All_Classes` 中
2. 系统的邮件服务（Brevo）是否正常运行

## 班主任与普通教师的区别

下表总结了班主任与同组非班主任用户的差异（仅列出有差异的部分）：

| 功能 | 班主任 | 非班主任（同组） |
|------|--------|----------------|
| 首页「班级FAD统计」卡片 | 可见 | 不可见 |
| `/fad/class-stats` 页面 | 可访问，自动显示本班数据 | 无法访问，自动跳转回首页 |
| 邮件通知 | 本班学生被记录时收到邮件 | 不会收到通知 |
| 其他所有功能 | 与同组用户完全一致 | 与同组用户完全一致 |

> **系统管理员和学生处老师的特殊情况：** 即使是班主任，系统管理员和学生处老师也可以通过「学校FAD统计」页面查看**任意班级**的统计数据（该页面有班级下拉选择框）。「班级FAD统计」是一个便捷入口，直接锁定你的班级，免去手动选择。

## 常见问题

### Q: 我是班主任，但首页看不到「班级FAD统计」卡片？

**可能原因：**
1. 你的登录邮箱与数据库中 `All_Classes` 表的 `HomeTeacherEmail` 字段不匹配
2. 数据库中尚未为你的班级配置 `HomeTeacherEmail`

**解决方法：** 联系系统管理员，请其检查并更新 `All_Classes` 中对应班级的 `HomeTeacherEmail` 字段。

### Q: 我同时是两个班级的班主任怎么办？

系统目前查询的是**第一个匹配的班级**。如果你需要查看另一个班级的数据：
- 系统管理员或学生处老师：使用「学校FAD统计」页面，手动选择班级
- 非学生处教师或任课老师：需联系管理员协助查看

### Q: 选择"学年"统计周期时，数据范围是什么？

学年 = 秋季学期 + 次年春季学期。例如 2025-2026 学年 = 2025年秋季 + 2026年春季。FAD 阈值（3次约谈、6次停课、9次劝退）和 Reward 冲抵均按学年范围计算。

### Q: 我能在班级统计页面直接执行 FAD 吗？

不能。班级FAD统计页面是**只读查看**页面。如需执行 FAD，请前往 **「FAD执行」** 页面操作。
