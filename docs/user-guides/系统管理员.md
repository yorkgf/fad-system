# 系统管理员操作指南

## 角色概述

系统管理员拥有系统全部权限，可访问所有页面、插入所有记录类型、执行和发放 FAD/Reward，以及管理教职工账号。

---

你可以看到以下功能卡片：

| 卡片名称 | 说明 |
|----------|------|
| **录入记录** | 添加学生行为记录 |
| **我的记录** | 查看已录入的记录 |

| **FAD通知单发放** | 发放纸质通知单（仅系统管理员可见） |
| **学校FAD统计** | 查看全校FAD数据统计 |
| **班级FAD统计** | 查看本班FAD数据统计（需为班主任） |
| **寝室表扬兑奖** | 兑换寝室表扬奖励（仅系统管理员可见） |
| **寝室清扫** | 记录寝室清扫情况 |
| **最佳寝室排名** | 查看优秀寝室排行 |

## 一、录入记录

> 首页点击 **「录入记录」** 卡片进入

此页面用于为学生创建各类行为记录。

### 1.1 可用记录类型

作为系统管理员，你可以插入所有 14 种记录类型：

**常规记录：**
- 早点名迟到、寝室迟出、未按规定返校、擅自进入会议室或接待室
- 寝室表扬、寝室批评、寝室垃圾未倒
- 21:30后交还手机(22:00前)、22:00后交还手机
- Teaching FAD Ticket、Teaching Reward Ticket

**管理员专属记录（仅系统管理员可见）：**
- **FAD** — 直接录入纪律处分
- **Reward** — 直接录入奖励
- **上网课违规使用电子产品** — 电子产品违规

### 1.2 基本操作步骤

1. **选择记录类型**：在「记录类型」下拉框中选择
2. **选择学期**：在「学期」下拉框中选择（默认为当前学期）
3. **选择学生**：
   - **寝室类记录**（寝室迟出、寝室表扬、寝室批评、寝室垃圾未倒）：先在「寝室」下拉框选择寝室号，系统自动加载该寝室学生，再在「寝室学生」下拉框勾选学生
   - **其他记录**：在「学生」搜索框输入姓名搜索，从下拉结果中勾选（支持多选）
4. **填写日期**：在「记录日期」日期选择器中选择（默认为今天）
5. **填写事由**：在「记录事由」文本框中输入原因（FAD、Reward、Teaching Ticket、上网课违规等类型**必填**）
6. 点击 **「提交」** 按钮

已选中的学生会显示为标签，点击标签上的 × 可移除。

### 1.3 特殊字段说明

| 记录类型 | 额外字段 | 说明 |
|----------|----------|------|
| FAD | **FAD来源类型**（单选按钮） | 必须选择：寝室类 / 教学类 / 电子产品 / 其他 |
| Reward | **优先冲抵执行**（开关） | 开启后优先冲抵FAD的执行状态，而非冲销整条记录 |
| Teaching FAD/Reward Ticket | **票据数量**（下拉框） | 可选 1~6 张，一次性为同一学生插入多张票据 |
| 上网课违规使用电子产品 | **取消资格时限**（单选按钮） | 选择"一个月"或"学期结束" |

### 1.4 累计规则与自动触发

提交成功后，系统自动检查该学生的累计记录数。达到阈值时页面会弹出提示框：

| 记录类型 | 累计阈值 | 自动触发结果 |
|----------|----------|-------------|
| 早点名迟到 | 2 次 | 自动生成 FAD（来源：其他） |
| 寝室迟出 | 2 次 | 自动生成 FAD（来源：寝室类） |
| 未按规定返校 | 2 次 | 自动生成 FAD（来源：寝室类） |
| 擅自进入会议室或接待室 | 2 次 | 自动生成 FAD（来源：其他） |
| 寝室批评 | 5 次 | 自动生成 FAD（来源：寝室类） |
| 寝室垃圾未倒 | 2 次 | 自动生成寝室批评（寝室批评再累计到 5 次会触发 FAD） |
| Teaching FAD Ticket | 3 张 | 自动生成 FAD（来源：教学类） |
| 22:00后交还手机 | 每次 | **立即**生成 FAD（来源：寝室类） |
| 寝室表扬 | 10 次 | 弹出提示可兑换 Reward（需到「寝室表扬兑奖」页操作） |
| Teaching Reward Ticket | 6 张 | 弹出提示可兑换 Reward（需到「教学票兑奖」页操作） |

> **注意：** 21:30后交还手机(22:00前) 不会触发 FAD，但 22:00后交还手机会立即触发。两者存储在同一个数据库集合中，请注意区分。

### 1.5 提交后的提示

- **全部成功**：显示绿色成功消息，并列出累计触发的 FAD/警告数量
- **部分失败**：显示橙色提示，说明哪些学生提交失败（例如寝室表扬/批评/垃圾未倒当天重复录入会被拒绝）
- **全部失败**：显示红色错误消息

## 二、我的记录

> 首页点击 **「我的记录」** 卡片进入

此页面用于查看、筛选、导出和撤回已插入的记录。

### 2.1 筛选栏

页面顶部有一排筛选控件，从左到右依次为：

| 筛选项 | 类型 | 说明 |
|--------|------|------|
| **记录类型** | 下拉框 | 选择要查看的记录类型，选择后自动刷新列表 |
| **FAD来源类型** | 下拉框 | 仅当记录类型为 FAD 时显示，可选：教学类/寝室类/电子产品/其他/未分类 |
| **学期** | 下拉框 | 选择学期，选择后自动刷新 |
| **教师** | 下拉框（可搜索） | 系统管理员和学生处老师专属，可按录入教师筛选 |
| **学生姓名** | 输入框 | 输入学生姓名进行筛选 |
| **班级** | 下拉框（可搜索） | 选择班级筛选 |
| **记录日期范围** | 日期范围选择器 | 选择开始和结束日期 |
| **导出CSV** | 绿色按钮 | 将当前筛选结果导出为 CSV 文件 |

### 2.2 记录列表

列表以表格形式展示，包含以下列：

| 列名 | 说明 |
|------|------|
| 学生 | 学生姓名 |
| 班级 | 所在班级 |
| 记录类型 | 记录的类型名称 |
| FAD来源类型 | 仅 FAD 记录显示 |
| 记录日期 | 格式 YYYY-MM-DD |
| 记录事由 | 鼠标悬停可查看完整内容 |
| 记录老师 | 录入该记录的教师 |
| 状态 | 彩色标签，含义见下表 |
| 操作 | 操作按钮 |

### 2.3 状态标签含义

| 记录类型 | 标签 | 颜色 | 含义 |
|----------|------|------|------|
| FAD | 已冲销 | 绿色 | Reward 已冲销该 FAD 记录 |
| FAD | 已执行未冲销 | 橙色 | FAD 已执行但未被 Reward 冲销 |
| FAD | 未执行 | 红色 | FAD 尚未执行 |
| 寝室批评 | 已累计FAD | 红色 | 该批评已触发 FAD 生成 |
| 寝室批评 | 未累计FAD | 蓝色 | 未达到累计阈值 |
| 寝室表扬 | 已累计Reward | 红色 | 该表扬已参与 Reward 兑换 |
| 寝室表扬 | 未累计Reward | 蓝色 | 尚未兑换 |
| 寝室垃圾未倒 | 已累计批评 | 红色 | 已触发寝室批评 |
| 寝室垃圾未倒 | 未累计批评 | 蓝色 | 未达到阈值 |
| 其他累计类 | 已累计FAD已发放 | 红色 | 已累计且 FAD 已发放 |
| 其他累计类 | 已累计FAD未发放 | 橙色 | 已累计但 FAD 未发放 |
| 其他 | 有效 | 蓝色 | 记录正常有效 |

### 2.4 操作按钮

#### 编辑来源（仅系统管理员可见，仅 FAD 记录）

点击 **「编辑来源」** 按钮后弹出对话框：
- 显示学生姓名和班级（只读）
- 「来源类型」下拉框可修改 FAD 来源类型
- 点击 **「确认」** 保存修改

#### 撤回记录

点击 **「撤回」** 按钮后弹出撤回确认对话框：

1. 页面顶部显示橙色警告提示，说明撤回后的影响
2. 如果该记录累计触发了其他记录（如 FAD、寝室批评），会显示**级联记录列表**，列出将被一并删除的关联记录
3. 在「撤回原因」文本框中输入原因（必填）
4. 点击红色 **「确认撤回」** 按钮执行撤回

**以下情况「撤回」按钮会显示为灰色不可点击**（鼠标悬停可看到原因提示）：

| 情况 | 提示信息 |
|------|----------|
| Reward 记录 | Reward记录不可撤回 |
| 已发放的 FAD | 已发放的FAD不可撤回 |
| 已冲销的 FAD | 已冲销的FAD不可撤回 |
| 已执行的 FAD | 已执行的FAD不可撤回 |
| 已发放的其他记录 | 已发放的记录不可撤回 |
| 已兑换 Reward 的寝室表扬 | 已兑换Reward的寝室表扬不可撤回 |
| 已兑换 Reward 的教学票 | 已兑换Reward的教学票不可撤回 |

### 2.5 分页

列表底部有分页控件，可选择每页显示 10 / 20 / 50 / 100 条记录。

## 三、FAD 执行

> 首页点击 **「FAD执行」** 卡片进入

此页面列出所有**未执行**的 FAD 记录，用于标记 FAD 已执行。

### 3.1 筛选栏

| 筛选项 | 说明 |
|--------|------|
| **选择学期** | 下拉框，选择后自动刷新列表 |
| **FAD来源类型** | 下拉框（可清除），选择后自动筛选 |

### 3.2 单条执行

在表格右侧「操作」列，点击 **「执行」** 按钮：
1. 弹出确认对话框，显示学生姓名
2. 点击 **「确认执行」** 完成操作
3. 该记录从列表中消失（已标记为已执行）

### 3.3 批量执行

1. 在表格最左侧勾选多条记录的复选框
2. 表格上方出现蓝色 **「批量执行 (N条)」** 按钮
3. 点击后弹出确认对话框，显示将要执行的记录数量和学生名单
4. 点击 **「确认执行」** 完成批量操作

> **注意：** FAD 来源类型列使用彩色标签区分：寝室类=橙色、教学类=红色、其他=蓝色。

## 四、FAD 通知单发放（系统管理员专属）

> 首页点击 **「FAD通知单发放」** 卡片进入

此页面列出所有**未发放**的 FAD 记录（已执行和未执行均包含），用于下载 PDF 通知单并标记为已发放。

### 4.1 筛选栏

与 FAD 执行页面相同：**选择学期** + **FAD来源类型**。

### 4.2 FAD 状态标签

列表中的「FAD状态」列显示当前状态：

| 标签 | 颜色 | 含义 |
|------|------|------|
| 已冲销 | 绿色 | 已被 Reward 冲销 |
| 已执行未冲销 | 橙色 | 已执行但未冲销 |
| 未执行 | 红色 | 尚未执行 |

### 4.3 下载通知单

点击 **「下载通知」** 按钮：
- 系统自动生成 PDF 通知单，包含学生姓名、班级、日期、原因等信息
- PDF 文件自动下载到本地

### 4.4 确认发放

**单条发放：** 点击 **「确认发放」** 按钮 → 弹出确认对话框 → 点击 **「确认」**

**批量发放：** 勾选多条记录 → 点击 **「批量确认发放 (N条)」** → 确认

> **注意：** 发放后记录不可撤回，请确认通知单已送达学生后再操作。

## 五、学校FAD统计

> 首页点击 **「学校FAD统计」** 卡片进入

### 5.1 筛选栏

| 筛选项 | 说明 |
|--------|------|
| **统计周期** | 选择春季/秋季/学年 |
| **班级** | 下拉框（可搜索），选择特定班级或查看全校 |
| **来源** | 可选全部/寝室类/教学类/电子产品/其他 |

### 5.2 数据概览卡片

页面顶部显示四个统计卡片：

| 卡片 | 说明 |
|------|------|
| **总FAD数** | 当前筛选条件下的 FAD 总数 |
| **已执行** | 已标记为执行的数量（绿色） |
| **已发放** | 已发放通知单的数量（蓝色） |
| **已冲销** | 被 Reward 冲销的数量（灰色） |

### 5.3 来源类型分布表

展示各来源类型的 FAD 数量分布，每行显示来源类型标签、总数、已执行数、已发放数、已冲销数。

### 5.4 未执行学生名单

当选择了特定班级时显示，列出该班所有有未执行 FAD 的学生，包含学生、班级、未执行数量（红色标签）、FAD 详情（最多显示 3 条，超出显示"更多"）。

### 5.5 学生FAD排名

展示 FAD 数量排名靠前的学生，点击行首 **展开箭头 ▶** 可查看该学生的 FAD 明细记录，包含日期、事由、老师、来源类型、执行状态、冲销状态。

来源分布列以彩色标签显示：寝室=橙色、教学=红色、电子产品=蓝色、其他=灰色。

## 七、Reward 发放（系统管理员专属）

> 通过首页或侧边导航进入

### 7.1 筛选

在「选择学期」下拉框中选择学期，列表自动刷新。

### 7.2 列表

显示所有**未发放**的 Reward 记录。「冲抵状态」列说明：

| 标签 | 颜色 | 含义 |
|------|------|------|
| 已冲抵FAD | 绿色 | 该 Reward 已用于冲抵 FAD |
| 未冲抵 | 灰色 | 尚未用于冲抵 |

### 7.3 操作

- **单条发放**：点击 **「确认发放」** → 确认对话框 → 点击 **「确认」**
- **批量发放**：勾选多条 → 点击 **「批量确认发放 (N条)」** → 确认

### 7.4 冲抵规则

Reward 的冲抵逻辑在插入 Reward 时自动触发：
- **2 个 Reward** → 冲抵最近一条 FAD 的执行状态
- **3 个 Reward** → 冲销最近一条 FAD 记录本身
- 冲抵按**学年范围**（秋季+次年春季）计算，不限单学期

## 八、寝室表扬兑奖（系统管理员专属）

> 首页点击 **「寝室表扬兑奖」** 卡片进入

### 8.1 页面说明

页面顶部有蓝色信息提示框，说明兑换规则：**累计 10 次寝室表扬可兑换 1 个 Reward**。

列表显示所有有寝室表扬记录的学生：

| 列 | 说明 |
|----|------|
| 学生 | 学生姓名 |
| 班级 | 所在班级 |
| 表扬次数 | 绿色标签（≥10）或灰色标签（<10） |
| 可兑换 | 显示可兑换的 Reward 数量（= 表扬次数 ÷ 10 取整） |
| 操作 | 两个按钮 |

### 8.2 操作按钮

**「打印奖励」**（蓝色按钮）：
1. 按钮在表扬次数 < 10 时为灰色不可点击
2. 点击后弹出确认框，显示学生姓名和将打印的 Reward 数量
3. 点击 **「确定打印」** 后系统生成 PDF 奖励证书并下载

**「标记已兑换」**（绿色按钮）：
1. 按钮在表扬次数 < 10 时为灰色不可点击
2. 点击后弹出确认框，显示将兑换的数量和消耗的表扬次数
3. 点击 **「确定兑换」** 后系统将 10 条表扬标记为已兑换，并生成对应 Reward 记录

> 首页点击 **「寝室清扫」** 卡片进入

### 9.1 页面说明

页面顶部有蓝色信息提示框说明规则。列表显示有「寝室垃圾未倒」记录的学生：

| 列 | 说明 |
|----|------|
| 学生 | 学生姓名 |
| 班级 | 所在班级 |
| 未清扫次数 | 红色标签（≥3）或橙色标签（<3） |
| 最近日期 | 最近一次违规日期 |
| 操作 | 确认按钮 |

### 9.2 操作

点击 **「确认清理」** 按钮（次数 < 3 时灰色不可点击）：
1. 弹出确认对话框，显示学生和累计次数
2. 点击 **「确认」** 后系统将未清扫记录转换为寝室批评

## 十、网课违规使用电子产品

> 首页点击 **「网课违规使用电子产品」** 卡片进入

此页面为**只读查看**，列出所有被暂停网课资格的学生：

| 列 | 说明 |
|----|------|
| 学生 | 学生姓名 |
| 班级 | 所在班级 |

## 重要注意事项

1. **发放操作不可逆**：FAD/Reward 一旦标记为已发放，该记录不可撤回。请确认已送达后再操作
2. **撤回的级联影响**：撤回一条基础记录（如早点名迟到）会同时删除由其累计生成的 FAD 或寝室批评
3. **FAD 来源类型修改**仅系统管理员可操作，建议在录入时就选择正确的来源类型
4. **学年范围**：FAD 阈值和 Reward 冲抵均按学年（秋季+次年春季）计算，不限单学期
5. **重复录入限制**：寝室表扬、寝室批评、寝室垃圾未倒**不允许同一学生同一天重复录入**
6. **22:00后交还手机** 每次录入都会立即触发 FAD；21:30后交还手机(22:00前) 不会触发
