# 非学生处教师操作指南

## 角色概述

非学生处教师拥有部门级管理权限，可执行 FAD、查看班级统计（仅限班主任）、查看电子产品违规和手机名单，以及管理日程。不可访问全校统计、寝室管理、数据查询、停课管理等校级功能。

---

## 首页 — 功能导航

登录后进入首页，你可以看到以下功能卡片：

| 卡片名称 | 说明 |
|----------|------|
| **录入记录** | 添加学生行为记录 |
| **我的记录** | 查看已录入的记录 |
| **FAD执行** | 管理FAD执行状态 |
| **班级FAD统计** | 查看本班FAD数据统计（仅班主任可见） |
| **网课违规使用电子产品** | 暂停网课资格名单 |
| **未交手机名单** | 查看未交手机学生 |
| **日程管理** | 预约教师面谈时间 |

**你看不到的卡片：** FAD通知单发放、学校FAD统计、寝室表扬兑奖、寝室清扫、最佳寝室排名、约谈/停课管理、教学票兑奖、数据查询

页面右上角点击用户名可展开菜单：**教师信息** / **修改密码** / **退出登录**。

---

## 一、录入记录

> 首页点击 **「录入记录」** 卡片进入

### 1.1 可用记录类型

- 早点名迟到、寝室迟出、未按规定返校、擅自进入会议室或接待室
- 寝室表扬、寝室批评、寝室垃圾未倒
- 21:30后交还手机(22:00前)、22:00后交还手机
- Teaching FAD Ticket、Teaching Reward Ticket

**不可插入：** FAD、Reward、上网课违规使用电子产品（系统管理员专属）

### 1.2 操作步骤

1. **选择记录类型**：在「记录类型」下拉框中选择
2. **选择学期**：在「学期」下拉框中选择（默认当前学期）
3. **选择学生**：
   - **寝室类记录**（寝室迟出、寝室表扬、寝室批评、寝室垃圾未倒）：先在「寝室」下拉框选择寝室号 → 在「寝室学生」下拉框勾选学生
   - **其他记录**：在「学生」搜索框输入姓名搜索，从结果中勾选（支持多选）
4. **填写日期**：在「记录日期」选择器中选择（默认今天）
5. **填写事由**：「记录事由」文本框（Teaching Ticket 类型必填）
6. 点击 **「提交」** 按钮

已选学生显示为标签，点击 × 可移除。点击 **「重置」** 清空表单。

### 1.3 特殊字段

| 记录类型 | 额外字段 | 说明 |
|----------|----------|------|
| Teaching FAD/Reward Ticket | **票据数量**（下拉框） | 可选 1~6 张 |

### 1.4 累计规则

提交成功后系统自动检查累计阈值，达到时弹出提示：

| 记录类型 | 阈值 | 结果 |
|----------|------|------|
| 早点名迟到 | 2 次 | 自动生成 FAD |
| 寝室迟出 | 2 次 | 自动生成 FAD |
| 未按规定返校 | 2 次 | 自动生成 FAD |
| 擅自进入会议室或接待室 | 2 次 | 自动生成 FAD |
| 寝室批评 | 5 次 | 自动生成 FAD |
| 寝室垃圾未倒 | 2 次 | 自动生成寝室批评 |
| Teaching FAD Ticket | 3 张 | 自动生成 FAD |
| 22:00后交还手机 | 每次 | 立即生成 FAD |
| 寝室表扬 | 10 次 | 提示可兑换 Reward（需联系管理员操作） |
| Teaching Reward Ticket | 6 张 | 提示可兑换 Reward（需联系管理员操作） |

> **注意：** 寝室表扬/批评/垃圾未倒不允许同一学生同一天重复录入。

## 二、我的记录

> 首页点击 **「我的记录」** 卡片进入

### 2.1 筛选栏

| 筛选项 | 说明 |
|--------|------|
| **记录类型** | 下拉框，选择后自动刷新 |
| **FAD来源类型** | 仅查看 FAD 记录时显示 |
| **学期** | 下拉框 |
| **学生姓名** | 输入框 |
| **班级** | 下拉框（可搜索） |
| **记录日期范围** | 日期范围选择器 |
| **导出CSV** | 绿色按钮，导出筛选结果 |

> **注意：** 非学生处教师不显示「教师」筛选框，只能查看自己录入的记录。

### 2.2 状态标签含义

| 记录类型 | 标签 | 颜色 | 含义 |
|----------|------|------|------|
| FAD | 已冲销 | 绿色 | 已被 Reward 冲销 |
| FAD | 已执行未冲销 | 橙色 | 已执行但未冲销 |
| FAD | 未执行 | 红色 | 尚未执行 |
| 寝室批评 | 已累计FAD / 未累计FAD | 红/蓝 | 是否已触发 FAD |
| 寝室表扬 | 已累计Reward / 未累计Reward | 红/蓝 | 是否已兑换 |
| 寝室垃圾未倒 | 已累计批评 / 未累计批评 | 红/蓝 | 是否已触发寝室批评 |
| 其他 | 有效 | 蓝色 | 记录正常 |

### 2.3 撤回记录

点击 **「撤回」** 按钮：
1. 弹出确认对话框，顶部橙色警告说明影响
2. 如有级联记录会列出将被一并删除的记录
3. 输入「撤回原因」（必填）
4. 点击红色 **「确认撤回」**

**不可撤回的情况**（按钮灰色，鼠标悬停显示原因）：
- Reward 记录不可撤回
- 已发放的 FAD/记录不可撤回
- 已执行/已冲销的 FAD 不可撤回
- 已兑换 Reward 的寝室表扬/教学票不可撤回
- **只能撤回自己录入的记录**（系统管理员或学生处老师录入的记录无法撤回）

## 三、FAD 执行

> 首页点击 **「FAD执行」** 卡片进入

列出所有**未执行**的 FAD 记录。

### 3.1 筛选

- **选择学期** — 下拉框
- **FAD来源类型** — 下拉框（可清除）

### 3.2 操作

**单条执行：** 点击 **「执行」** → 确认对话框 → **「确认执行」**

**批量执行：** 勾选多条 → **「批量执行 (N条)」** → 确认

来源类型标签：寝室类=橙色、教学类=红色、其他=蓝色。

---

## 四、班级FAD统计（仅班主任）

> 首页点击 **「班级FAD统计」** 卡片进入

> **注意：** 此卡片仅在你担任班主任时显示。如果你不是班主任，首页不会出现此卡片。

### 4.1 页面内容

系统自动加载你所带班级的数据，无需手动选择班级。

- **数据概览**：四个统计卡片 — 总FAD数、已执行、已发放、已冲销
- **来源类型分布表**：各来源类型的 FAD 数量
- **未执行学生名单**：列出有未执行 FAD 的学生
- **学生FAD排名**：点击 **▶ 展开** 查看明细（日期、事由、老师、来源、状态）

### 4.2 筛选

| 筛选项 | 说明 |
|--------|------|
| **统计周期** | 春季 / 秋季 / 学年 |
| **来源** | 全部 / 寝室类 / 教学类 / 电子产品 / 其他 |

---

## 五、网课违规使用电子产品

> 首页点击 **「网课违规使用电子产品」** 卡片进入

只读列表，显示被暂停网课资格的学生：

| 列 | 说明 |
|----|------|
| 学生 / 班级 | 基本信息 |
| 违规日期 | 违规发生日期 |
| 取消资格至 | 红色=仍在暂停期，灰色=已过期 |
| 违规事由 | 鼠标悬停可看完整内容 |
| 记录老师 | 录入老师 |
| 状态 | **禁止上课**（红色）或 **已恢复**（绿色），自动判断 |

## 六、未交手机名单

> 首页点击 **「未交手机名单」** 卡片进入

显示**当天**手机交还记录。

| 列 | 说明 |
|----|------|
| 学生 / 班级 | 基本信息 |
| 迟交类型 | 橙色 "21:30后(22:00前)" 或 红色 "22:00后" |
| 记录老师 | 录入老师 |
| 是否触发FAD | 红色 "是"（22:00后）或 灰色 "否"（21:30后） |

点击 **「刷新」** 按钮更新数据。仅显示当天记录。

## 七、日程管理

> 首页点击 **「日程管理」** 卡片进入

### 7.1 创建家长会时段

1. **选择日期**：日期选择器选择多个日期（已过日期不可选）
2. 勾选 **上午时段** 和/或 **下午时段**，设置开始和结束时间
3. **时间分割**：选择 10 / 15 / 20 / 30 分钟
4. **地点**（必填）：选择"线下"或"腾讯会议"
5. **备注**：可选
6. 点击 **「创建时段」**

> 如果新时段与已有时段冲突，系统会弹出错误提示并列出冲突详情。

### 7.2 管理时段

- 未预约的时段可点击红色 **「删除」** 删除
- 已有预约的时段显示 **「已有预约」** 标签，不可删除
- **「删除全部未预约」** 批量删除（需确认，不可恢复）
- **「我的日程」** 按钮进入日历视图

## 八、设置

### 修改密码

> 右上角用户名 → **「修改密码」**

1. 输入「当前密码」
2. 输入「新密码」（≥ 6 位）
3. 输入「确认新密码」（须一致）
4. 点击 **「确认修改」**

修改成功后自动退出登录。

### 教师信息

> 右上角用户名 → **「教师信息」**

查看和编辑个人信息。

---

## 重要注意事项

1. **只能撤回自己录入的记录**，管理员录入的记录无法撤回
2. 你可以执行 FAD，但**不可发放** — 需联系系统管理员
3. 班级FAD统计仅班主任可见，非班主任看不到此功能卡片
4. 寝室表扬兑换 Reward、教学票兑奖、停课管理等需联系管理员
5. 撤回记录会级联删除由该记录累计生成的 FAD/寝室批评
6. 寝室表扬/批评/垃圾未倒不允许同一学生同一天重复录入
