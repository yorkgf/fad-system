
# NYU Pre-College Online — 纽约大学预科项目（线上）


## 概述

NYU Pre-College允许高中生修读纽约大学本科课程并获得大学学分，课程由世界一流的NYU教师授课，提供线上和校园两种模式。涵盖艺术媒体、STEM、人文社科、商科等多个领域。

## 关键信息

| 项目 | 详情 |
|------|------|
| **学分** | 是，可获得大学学分 |
| **授课形式** | 线上 + 校园 |
| **适合对象** | 高中生（2026/2027/2028年高中毕业者） |
| **年龄要求** | 2026年6月20日前满16岁，2026年7月31日前不满19岁 |
| **2026状态** | 申请已开放 |

## 学分

- 每门课程 **4学分**
- 可获得正式成绩单
- 学分可用于未来大学学位

## 费用

| 项目 | 费用 |
|------|------|
| **申请费** | $75（不可退还） |
| **4周校园（4学分）** | $9,100 |
| **7周校园（8学分）** | $15,735 |
| **7周远程/走读（4学分）** | $4,180 |
| **7周远程/走读（8学分）** | $8,160 |

> 线上/走读路线显著低于校园住宿路线。

## 申请要求

- 2026/2027/2028年高中毕业
- 年龄要求（16-18岁）
- 每位申请者只能申请一个方向
- 需要展现强烈学术动机
- 录取考虑学术准备度、成熟度和兴趣

## 学术支持

- 个人学术指导（Personal Academic Advising）
- College 101系列：大学申请、学习技能、助学金信息
- 可使用NYU资源、活动和服务

## 国际学生

- 接受来自世界各地的学生
- 校园项目可提供签证文件
- ⚠️ **机构资助签证的学生不能修读在线课程**
- 纯线上课程不涉及签证要求

## 助学金

- 有限资金帮助有经济困难的学生
- 通常仅覆盖部分学费
- ⚠️ **助学金仅限美国公民和永久居民**
- 早申提高获得资助的机会

## 备注

- 线上路线性价比最高（$4,180/4学分 vs 校园$9,100）
- 建议尽早申请以获得优先录取和资助考虑
- 课程涵盖领域广泛，从STEM到艺术到商科
