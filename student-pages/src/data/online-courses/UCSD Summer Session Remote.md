
# UCSD Summer Session Remote — UCSD暑期课程（远程）


## 概述

UC San Diego暑期课程面向所有学生开放（"Anyone can take classes at UC San Diego during the Summer!"），提供面授和远程两种模式，高中生、新生、转学生均可申请，可获得UCSD学分。

## 关键信息

| 项目 | 详情 |
|------|------|
| **学分** | 是，UCSD正式学分 |
| **授课形式** | 面授 + 远程（Remote Options） |
| **适合对象** | 高三学生、准大一新生、转学生、研究生、访问学生、校友 |
| **2026状态** | 已开放 |

## 2026年暑期会期

| 会期 | 日期 |
|------|------|
| Summer Session 1 | 2026/6/29 — 8/1 |
| Summer Session 2 | 2026/8/3 — 9/5 |
| Special Session | 2026/6/16 — 9/19 |

## 课程信息

- 通过课程表（Schedule of Classes）和课程预览（Preview of Classes）页面查看可选课程
- 远程课程（Remote Courses）有专门入口
- 可查询学分单位（Units）信息

## 申请流程

1. 选择学生类型（Visiting Students等）
2. 查看课程安排
3. 通过招生门户提交申请
4. 联系 summer@ucsd.edu 获取帮助

## 费用

- 具体学费需查看 Tuition and Fees 页面
- 可申请暑期助学金（Financial Aid）
- 有账单和缴费页面（Billing Statements, Fee Payment）
- 逾期缴费会产生滞纳金（Late Fee Penalties）

## 住宿

- ✅ 暑期住宿对所有符合条件的学生开放

## 联系方式

- Email: summer@ucsd.edu
- 工作时间: 周一至周五 9:00 AM — 4:00 PM

- UCSD在US News排名约#33，略出前30但仍属顶尖公立大学
- 申请门槛相对友好，"Anyone can take classes"
- 远程课程的具体要求和费用需通过课程表查看
