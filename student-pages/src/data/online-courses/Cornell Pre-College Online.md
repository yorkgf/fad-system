
# Cornell Pre-College Online — 康奈尔大学线上预科项目


## 概述

康奈尔大学暑期学院（School of Continuing Education）提供的线上预科项目，允许高中生修读康奈尔本科课程并获得**可转学分**。课程由康奈尔教职员工教授，夏季和冬季均有开设。

## 关键信息

| 项目 | 详情 |
|------|------|
| **学分** | 是，正式康奈尔成绩单，学分通常可转至康奈尔或其他大学 |
| **授课形式** | 完全在线 |
| **授课时间** | 夏季（3周或6周）和冬季 |
| **开设学期** | 2026年夏季申请已开放 |
| **适合对象** | 即将升入高三或高二的高中生 |

## 课程选择

- 数十门常规康奈尔本科在线课程
- 涵盖多个学科领域
- 与本科生同步修读
- 使用 Canvas 学习管理系统
- 作业有固定截止日期

## 学分与成绩

- 获得**正式康奈尔大学成绩单**
- 学分一般可转至康奈尔或其他高校
- 夏季课程可选 **3-8学分**

## 费用详情

| 项目 | 费用 |
|------|------|
| **每学分学费** | $1,940 |
| **申请费** | $75（不可退还） |
| **3周3学分** | $5,820 |
| **3周4学分** | $7,760 |
| **6周6学分** | $11,640 |
| **6周7学分** | $13,580 |
| **6周8学分** | $15,520 |

> 在线学生仅收取学费。课程材料可能产生额外费用。

## 申请要求

- 即将升入高三或高二（Junior/Senior）
- 2026年夏季截止日期：**最早至 2026年4月28日**
- 先到先得，课程满员即止
- 通过招生门户网站提交申请

## 国际学生

- ✅ **接受国际学生**
- ⚠️ 受制裁地区（伊朗、朝鲜、叙利亚、古巴、俄罗斯、乌克兰克里米亚/顿涅茨克/卢甘斯克）的学生须获得康奈尔出口管制合规官员书面批准方可注册

## 链接

- 官网：https://sce.cornell.edu/precollege
- 申请入口：通过官网 "Start an Application"
- 联系方式：通过官网招生页面

## 备注

- 课程名额有限，申请建议尽早
- 有助学金（Financial Assistance）页面，但具体国际学生资助信息不详
