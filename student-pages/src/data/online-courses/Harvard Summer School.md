
# Harvard Summer School — 哈佛大学暑期学校


## 概述

哈佛大学暑期学校提供300+门大学学分课程，面向高中生和本科生开放，包含校园和线上两种模式。高中生可选择2周预科项目或4/7周灵活中学项目，由哈佛教师授课。

## 关键信息

| 项目 | 详情 |
|------|------|
| **学分** | 是，哈佛正式成绩单 |
| **授课形式** | 校园 + 线上 |
| **适合对象** | 高中生、大学生 |
| **2026状态** | 注册已开放 |

## 高中生项目

### 2周预科项目（2-Week Pre-College Program）
- 体验大学级别课程
- 与来自全球的同龄人一起学习
- 校园住宿或在线学习

### 4/7周灵活中学项目（Secondary School Program）
- 修读1-2门哈佛本科课程（每门4学分）
- 与哈佛在校生和来自世界各地的学生共同学习
- **线上路线**：可在家远程修读1-2门4学分课程
- **校园路线**：住宿于哈佛校园

## 学分

- 每门课程 **4学分**
- 获得**哈佛正式成绩单**
- 显示课程成绩（"requirements met" 或 "requirements not met"）

## 费用

| 项目 | 费用 |
|------|------|
| **申请费** | $75（不可退还） |
| **2周预科（校园）** | $6,100/次（含住宿、餐饮、项目、医保） |
| **7周校园（8学分）** | $15,735 |
| **7周远程/走读（4学分）** | $4,180 |
| **7周远程/走读（8学分）** | $8,160 |

## 2026年申请截止日期

| 节点 | 日期 |
|------|------|
| 早审 & 优先资助 | 2026年1月7日 |
| 常规截止 | 2026年2月11日 |
| **最终截止** | **2026年4月1日** |

## 申请要求

- 2026/2027/2028年高中毕业并进入大学
- 年龄要求：2026年6月20日前满16岁，7月31日前不满19岁
- 录取考虑：**学术准备度、成熟度、学术兴趣和动机**

## 线上项目

- 与暑期项目同步开设虚拟路线
- 可在家修读1-2门4学分课程
- 无需校园住宿
- 由哈佛教师授课

## 国际学生

- 校园项目提供签证文件
- ⚠️ **机构资助签证的学生不能修读在线课程**
- 纯在线课程无需签证
- 申请者需自行了解移民相关规定

## 助学金

- 有限资金帮助有经济需要的学生
- ⚠️ **仅限美国公民和永久居民**
- 通常仅覆盖部分学费
- 早申提高获得资助的概率
- 所有授课模式均可申请

## 学术支持

- College 101：大学申请、学习技能、财务规划
- 个人学术指导
- 可使用哈佛学术资源

## 备注

- 哈佛是US News #3顶尖学府，竞争最为激烈
- 4月1日为最终截止日期，但建议尽早申请
- 线上路线性价比最高（$4,180/4学分）
- 国际学生如已有美国学校签证，不能选在线课程
