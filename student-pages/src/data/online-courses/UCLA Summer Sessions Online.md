
# UCLA Summer Sessions Online — UCLA暑期课程（线上）


## 概述

UCLA暑期课程提供800+门课程，包含完全在线和混合（面授/在线）两种模式，面向高中生、大学生和国际学生开放，可获取UCLA学分。

## 关键信息

| 项目 | 详情 |
|------|------|
| **学分** | 是，UCLA正式学分 |
| **授课形式** | 完全在线 或 混合（in person/online） |
| **适合对象** | 高中生、大学生（Visiting College Student）、成人/职业学习者 |
| **2026状态** | 注册已开放，通过 portal.summer.ucla.edu/apply/ 申请 |

## 课程领域

- 商科（Business）
- 建筑（Architecture）
- 电影（Film）
- 800+课程可选

## 国际学生要求（I-20/F-1签证）

### 可申请I-20的项目
- Summer Courses
- Summer Intensives
- Summer College Immersion Program
- 大多数 College/Professional Summer Institutes

### 不可申请I-20的项目
- **仅在线**（Exclusively Online）项目 ❌
- Summer Precollege Focus Series ❌

### 英语要求（两年内有效）

| 考试 | 最低分数 |
|------|---------|
| TOEFL (iBT) | 80 |
| IELTS | 6.5 |
| Duolingo | 105 |
| iTEP | 5.0 |
| TOEIC | 685 |

**豁免国家**: 澳大利亚、加拿大、爱尔兰、新西兰、新加坡、英国

### I-20申请材料
- 全日制注册
- 有效护照照片页
- 英语能力证明
- 财力证明（生活费估算：1-4周$1,000，5-8周$2,000，9-12周$3,000）

### I-20截止日期
- 8月5日前开始的项目：**5月1日**
- 8月5日后开始的项目：**6月1日**

### 全日制注册要求
- 课程数量 ≥ 会期周数（如A6会期至少选6学分）
- 连续会期之间不能有间隔
- 在线课程不得超过总学分的50%

### 医疗保险
- I-20学生**必须**购买保险，不可放弃
- 自动注册 Summer BruinCare 和 GeoBlue Blue Cross/Blue Shield

### 额外费用（I-20学生）
- 服务费（Service Fee）
- iStart费
- BruinCare医疗保险

> 纯在线课程学生**无需支付**上述额外费用。

## 费用

- 网站提供费用计算器（Fee Calculator）估算费用
- 加州高中生有奖学金机会
- 多种付款方式（Flywire、Alipay等，需提前1-2周处理）

## 备注

- 纯在线课程不提供I-20，但国际学生可以在本国直接注册在线课程
- 如需获得F-1签证到校上课，需满足全日制注册和在线≤50%的限制
