# 特定主题技巧

> 针对特定主题（challenge、family、career等）的写作技巧

## L043: 谨慎写困难话题 / Write about Difficult Topics Wisely

**核心要点：** 如果写困难话题，确保你有正确的基调并展示了成长。

我知道他们总是说不要写悲剧事件或'悲情故事'，因为每个人都这样做，招生读者不会可怜你。

然而，我觉得你应该写一个对你独特的经历（对我来说，这是我第一次作为演员在音乐剧的舞台上演出，而不是在管弦乐队演奏或作为舞台工作人员工作）。

还有，讲一个故事！如果你享受你的经历并被它启发，你的写作可能会反映那个情感方面并揭示对你来说新和有趣的东西。招生官也会了解你很多！

如果你选择写一个困难话题，确保你展示如何从中成长，而不仅仅是抱怨或寻求同情。

## L045: 写家庭关系 / Write about Family Relationships

**核心要点：** 家庭关系可以是很好的文章题目，但要独特地写。

我写了关于我的家族和我的遗产的最爱的文章（用来申请众多大学的文章之一）。我一直为我的遗产感到自豪，并在我非常支持的家庭中长大，这在我整个童年对我影响很大。

我被告诉写一个你充满激情并且对你重要的事情是理想的，既然我对自己的文化的爱总是给我对其他文化的很大欣赏并让我的眼睛以不同的方式向世界打开，我觉得这是完美的题目。

我使用了我小时候从堂表那里学到的黎巴嫩游戏这个具体想法来展示我的遗产如何影响我。父母可以成为学生生活中最有影响力的人之一，这是很多学生的文章是这个题目有道理。

## L046: 写课外活动 / Write about Extracurricular Activities

**核心要点：** 课外活动是很好的文章题目，但要专注于具体经历而非列表。

我最喜欢的一个建议是选择一个对我作为一个人极其重要的题目。我认为写那些真正有意义并且能注入文章自己人格要素的题目，对专业招生官来说，很容易发现虚假。

然而，当申请者可以写一个实际有意义的题目并能注入文章一些他或她自己的性格要素时，它会在坐在招生官桌子上的那堆申请中脱颖而出。

最终，我相信这就是大学申请文章的目的，尤其是对于加州大学学校。文章是你给标准申请带来人面孔和个性的机会。我写了关于我的遗产作为韩国美国人，因为它谈到了我是谁、我现在是谁以及我想成为谁。

## L047: 写挑战 / Write about Challenges

**核心要点：** 写你面临的挑战，但专注于你如何克服和从中学习。

在写作过程中，我意识到我有一个年轻身心，经历了一些任何人可能经历的最困难经历。好几年，这是一个斗争来面对、接受和应对这些经历的现实。

写文章帮助我意识到，在我的生活中第一次，我为遇到的逆境感到自豪，而不是被羞辱、内疚和脆弱消费；这三个属性是带来改变的属性。

然而，最大的实现之一是视角的力量可以开始平息人们之间的一些差异。我挑战自己，不仅分享我的成就，而是解释我为什么完成它——我的动机是什么。

任何人都可以做事情，但真正让你从人群中脱颖而出的是你的旅程——你为什么做你所做的步骤和理由。

## L048: 写职业目标 / Write about Career Goals

**核心要点：** 展示你对职业的热情以及你的经历如何塑造这些目标。

对于开放文章题目，我讨论为什么我选择我的研究领域以及为什么我申请加州大学系统而不是加州州立大学系统。

我知道审阅者会想知道什么激励了我，因为这会决定我如何通过我的上学以及我在大学毕业后用我的教育做什么。

我选择这个题目因为它允许我在一个简明的两页文件中传达我生活的许多不同方面，并且不会感觉重复我文章中的信息。此外，我可以传达关于我个人信仰、我的家庭、我的背景和我的目标的信息，所有这些都会让读者觉得我更凝聚和有趣。
