## L005: 早期开始 / Start Early

**核心要点：** 不要等到截止日期前夜才开始写，好文章是改出来的。

几乎每个学生都做过——等到最后一刻才写论文或做项目。有时结果还可以，但有时就不行了。写大学申请文章时拖延是不明智的。

这需要时间。即使你能在截止日期前夜写出文章，还是最好不要。好文章需要时间'发酵'。作者写完后，把它放一段时间，然后用新鲜的心态回来继续。

提前开始这样你有充足的时间思考和打磨你的文章。提前开始也是唯一的方法来确保你有足够的时间来修改和完善。

## L013: 彻底校对 / Proofread Thoroughly

**核心要点：** 避免拼写、语法和标点错误，确保文章的清晰度。

有些学生在文章中写错了大学名字，这是一个可以通过校对避免的错误。更多学生有拼写、语法或标点错误。

虽然这些类型的错误通常不会完全破坏你的申请，但它们可能会让人分心，最多是向招生官表明你粗心，不是认真对待他们的大学。

避免这一点不仅要使用你电脑的拼写检查，还要请别人帮助校对你的文章。两次更好。使用拼写检查，但要记住计算机不聪明，所以你必须自己仔细校对你的文章。

确保你提交的文件中没有像"哈佛"、"耶鲁"这样的错误。

## L014: 获取反馈 / Get Feedback

**核心要点：** 让别人阅读你的文章并提供建议，但不要听从每个人的意见。

实际上你在杂志、书、报纸或互联网上读到的每一篇文章都编辑过。原因是写作不应该是一个孤立的经历。

你可能确切知道你想在自己的脑海中传达什么，但当你把它放在纸上时，它可能不会像你在脑海中那么清楚地出来。获取反馈有帮助。

请父母、老师甚至朋友阅读并评论你的文章。他们可以帮助你识别什么可以被编辑出去，什么需要更好地解释，或者你可以如何改进你的作品。

但不要听从每个人的意见。一两个人给出文章反馈就足够了。如果你问超过那个，你可能会失去你写作的焦点。有太多编辑者会稀释你的作品，因为每个人都有不同的观点。

## L031: 让你的文章发展 / Let Your Essay Evolve

**核心要点：** 不要害怕改变或随着你进一步进入写作过程让文章的主题发展。

我可以提供的最好建议是不要害怕随着你进一步进入写作过程而改变或让文章的主题发展。

当我开始为文章头脑风暴时，我相信我会写的想法和主题与我实际写的有明显不同；这是我整个写作过程中不断调整的结果。

我在开始时担心，如果我开始写只与我主要主题切线相关的想法，我会搞砸我的整个文章。但尽管这些最初的保留，我意识到随着我在文章草稿之后草稿的过程中工作，让文章的轨道改变形状是可以的。

最终产品结果变成了与我预期的不同的东西，但我对最终结果感到满意。

## L032: 测试多篇文章 / Test Out Multiple Essays

**核心要点：** 测试多篇文章，因为可能是一个好主意但不会成为一篇好文章。

测试多篇文章，因为有时可能是一个好主意但不会成为一篇好文章。

永远不要觉得被你开始的东西困住。申请读者永远不会知道你在最终产品前删除了多少草稿。

下一个提示是给定的。尽早开始。真的。下一个提示。一遍又一遍编辑它，让尽可能多的人读它，接受他们的所有意见但不要觉得你必须听从所有意见。

这是你的文章，你想用它做什么就做什么，但要记住你想让申请读者也喜欢它！

关于你的文章内容，也许你开始了一个个人故事。不要只是写它——每个人都有很棒的故事可讲！你必须通过把它与你的性格特质、你的课外活动、你未来可能想完成的事情（可能在他们学校……）联系起来而脱颖而出。

## L040: 创建成就列表 / Create a List of Accomplishments

**核心要点：** 创建一个成就列表，帮助你记忆你的所有经历。

另一个好主意是创建一个成就列表，从最早二或三年级开始。这在填写申请的奖项/荣誉部分时很方便，虽然事情真正从九年级开始"计算"。

除了成就之外，我有我加入的每一个俱乐部、我在每一个比赛中赢得的每一个奖项、我在俱乐部中担任的每一个领导职位等等。

这听起来可能很牵强，但在申请季节的繁忙中忘记你所做的一切很容易。有列表方便并基于它工作要容易得多。

## L049: 使用拇指测试 / Use the Thumb Test

**核心要点：** 确保你的文章足够个性化，只有你可以写。

在你写文章时，把你的拇指放在你的名字上。你能否把另一个名字放在上面因为它可能是很多其他学生写的文章？

或者文章对你如此个性化以至于基本上你的名字是唯一可以放在上面的名字？如果你没通过拇指测试，是时候重新思考题目或你处理它的方式。

你想让你的文章对你独特。

## L050: 从写作过程中学习 / Learn from the Writing Process

**核心要点：** 写作过程是一个自我发现和反思的机会。

我从写文章中学到很多比我开始时预期的更多。像大多数经历一样，它直到被反思处理后才完全被欣赏和理解。

写我的文章，这是关于一次去科学博览会比赛的旅行，给了我机会——甚至可能强迫我——反思围绕在我当时的事件和活动。

我不仅认识到那次旅行在我高三年的更广泛背景中的作用，而且我还在几个看似次要的事件中发现了一些宝石。

让自己无法与我的经历分开——我只是发现自己用精细的镜头重新讲述我的故事。这个新视角教会了我很多关于我自己。我还认识到友谊，尽管它们看起来短暂，如何影响我。

旅行教会我，真正热情的力量可以胜过身体残疾。
