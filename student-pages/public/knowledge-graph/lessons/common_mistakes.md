# 常见错误

> 应该避免的错误和雷区

## L016: 不要用噱头 / Don't Use Gimmicks

**核心要点：** 避免使用奇怪的花招来吸引注意力，这会让招生官觉得你不合适。

申请者曾被知道在文章中附上一只鞋，并附上一张纸条写着"现在我一只脚已经进门了"。他们还用不同的字体和颜色打印他们的文章，发送礼物或食物甚至包括情绪音乐，旨在为招生官读文章时设置情绪。

几个学生甚至发送现金！

虽然像这样的噱头可能会吸引一些注意力，它们对申请者没有太大帮助，尤其是那些发送现金的少数人——这是一个明确的禁止。

确实你想让你的文章脱颖而出，但不是以招生官认为你不合适或仅仅愚蠢的方式。如果你有创意东西的想法，先给老师或顾问看看他们怎么想。

## L017: 不要越界 / Don't Cross the Line

**核心要点：** 不要分享过于个人或不当的信息。

有些学生把分享关于自己的信息这些建议记在心上，但他们最后分享太多。他们认为他们必须如此揭示，以至于他们用文章来承认他们永远不会以其他方式承认的事情。

有学生写过醉酒、感到自杀或给老师恶作剧。在正确的情况下，这些题目可能行得通。例如，如果恶作剧是轻松愉快的，他们的老师对此有好的幽默感，这是可以接受的。

但大部分情况下，这些类型的题目高度冒险。确定你是否越界的最好方法是与一两个成年人分享你的想法并获得他们的反应。

## L018: 不要重复申请表信息 / Don't Repeat the Application

**核心要点：** 文章应该传达新的信息，而不是重复申请表中已有的内容。

文章不是申请表，也不是简历。换句话说，文章是你拥有的最好机会，要么深入讨论你在申请表中写的某件事，要么详细阐述在申请表中并不真正适合的新事物。

重复申请表中已经有的东西对你没有帮助。文章不应成为你的简历。文章是传达个人声音的动态叙述，有潜力解释性格。相反，简历和问卷可能是独特的，因为没有人有你完全的技能组合或经历范围，但语言是干枯和静态的。
