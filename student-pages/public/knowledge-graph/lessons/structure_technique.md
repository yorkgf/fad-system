# 结构技巧

> 文章结构、开篇、结尾等技巧

## L003: 小处着手，讲好一个故事 / Start Small - Tell One Story

**核心要点：** 不要试图写你整个一生，用一个具体的故事或经历来展示你的性格。

500到800字的文章没有空间让你写自出生以来最重要的10个成就，或者在欧洲为期三周的暑期项目中所做的一切。

相反，这个空间大概只能容纳一到两个成就，或者暑期项目中的一到两个经历。不要试图分享你的整个人生，分享我们所说的你生活的"一片"。

这样做可以给文章焦点，让你有空间更深入地覆盖这个题目。一个具体的故事、一个细节，比宏大叙事好得多。

不说"我一生都热爱科学"，而说"我十岁那年拆了家里第一个收音机……"。

## L006: 回答题目 / Answer the Prompt

**核心要点：** 确保你的文章真正回答了题目，不要只写你想要说的内容。

虽然这看起来很简单，但很多学生就是不听这个建议。这个建议尤其适用于那些重复使用文章的学生。

我们高度推荐重复使用，因为这样可以为你节省时间，用一篇文章用于多所大学，但警告是你需要编辑文章使其回答所问的问题。

当学生提交一篇文章——即使是一篇写得很好的文章——但不回答题目时，这会让招生官感到厌烦。他们认为学生要么不够认真到提交一篇专门为那所大学写作或至少编辑过的文章，要么就是他们不听指示。

无论哪种情况，那都不是你想要留下的印象。

## L009: 强有力的开篇 / Have a Strong Opening

**核心要点：** 好的开篇能立即吸引招生官的注意力。

学生开始文章时重复问题甚至陈述他们的名字。这对吸引招生官的注意力没有什么帮助。

当然，他们会读完全文，但总是有一个好的开始有帮助。想想你如何描述你处于的情况，传达你强烈相信的事情或分享一个可能意想不到的轶事。

开篇不会成就或破坏你的文章，但它可以让你朝正确的方向开始。招生官每天读上千篇文章，如果你不能用第一句话吸引他们的注意力，他们可能不太愿意继续读。

让他们享受你的文章，这绝对会回报。

## L010: 聚焦一个想法 / Focus on One Idea

**核心要点：** 一篇好的文章应该有一个单一、深入思考的信息，而不是很多不完整的想法。

有一个单一、深入思考的信息比很多不完整的更好。聚焦允许你深入探讨一个具体题目并为你的立场提出强有力的论证。

有说服力地写。你可以用例子来说明你的观点。这种类型的文章很难写，因为可能感觉像是你试图把整个生活挤进几段——这通常导致太多未充分发展的想法，都不能公正地对待你。

选一个想法并坚持它。这样做会给你创造性思考的灵活性，并帮助你觉得你控制了自己的写作。试着不要遵循任何你认为招生官在寻找的公式。你应该写一些个人的东西，描绘你作为一个人的形象。

## L020: 写之前先思考 / Think Before Writing

**核心要点：** 花和实际写作一样多的时间思考你要写什么。

你应该花和实际在纸上写词一样多的时间思考你要写什么。这将帮助你淘汰那些走不通的题目，确定哪个题目对你有最大的吸引力，并弄清楚你想说的确切内容。

自己大声说出来文章或与父母、老师或朋友讨论你的想法会有帮助。另一个人可能会看到一个角度或缺陷，而你没有。

让想法自然地首先出现。我发现我最好的想法不是当我坐在桌前试图写作时出现的，而是在白天思考我的环境和生活中的时候。

## L025: 与申请表有联系 / Connect with the Application Form

**核心要点：** 文章和申请表之间应该有一定的连续性，而不是完全重复。

虽然你不想在文章中逐字逐句重复申请表的信息，但文章和你的申请表之间有一些连续性通常是好主意。

如果你在文章中写你生命中最热爱的是弹钢琴以及你每周花10小时练习，这个爱好应该在申请表中提到，连同任何你给过的演出或你赢得的奖项。

在文章中写你热爱一个活动而在申请表中没有任何提到它，这没有意义。记住招生官正在看你的申请作为一个整体，他们应该通过所有部分——包括申请表、文章、成绩单、推荐信和面试——有一个完整和凝聚的你。

## L028: 在好心情时写作 / Write When in a Good Mood

**核心要点：** 当你感觉饱腹、温暖和健谈时写作，不要在不适合的时候写作。

好心情时写作。不要在11:30PM时写作，文章30分钟后12:00午夜就到期。不要在你刚刚考试失败时写作。不要在你刚刚和父母吵架（推测是关于为什么你还没有完成大学申请）后写作。

在你感觉饱腹、温暖和健谈时写作。突然的想法让你想Ooooohhh!!时，这可能意味着你必须诉诸我的怪方法带着笔记本大约一周（是的，我知道这很怪；很多人告诉我这很怪），但这绝对值得努力。

## L029: 从自由写作开始 / Start with Free Writing

**核心要点：** 从自由写作开始，帮助找到你的最佳想法。

我最好的建议是尽可能早地开始写作。尽早开始，你就有更多时间改进你的文章。也，请你的英语老师帮助看你的文章。

我也建议尽早开始写作。你会更放松和满足。我会说首先自由写作，那总是我得到最好想法的方式。

从一张空白纸（或空白word文档）开始——然后连续写作五到十分钟，不擦除任何东西并且永远不停止词流。一旦你找到一个题目，写一个草稿——然后把它放一边几天。只有时间的视角（你会想要拥有的奢侈品）才能让你真正编辑得好。

## L033: 回答题目所问的 / Answer What the Prompt Asks

**核心要点：** 确保你的文章回答了题目所问的内容。

确保回答题目！不要只是谈论你的品质；讨论这些品质将如何有助于你的成功、大学和你最想影响的世界的一部分。

确保你确实在回答题目！当你开始写作文章时，坐下来想想你自己——你在高中做了什么，什么是最重要的经历，什么对你的性格发展贡献最大？

不要害怕写一个"常见"主题，但要确保以一个对你独特的方式写。当你选择题目时，确保它是有深度的，不要只是一个浅层的东西。
