## 使用建议

### 按主题找范文
需要写"挑战"主题 → 查看 `IvyLeague/challenges-*.md`

### 按学校找范文
需要哈佛补充文书 → 查看 `NiuXiaoNiuwen/text00005.md` 中 Harvard 部分

### 按题目类型找
需要UC PIQ题目参考 → 查看 `UC/essays/Chapter001.md` 错误分析 + `Chapter002-*.md` 范文

### 学习写作技巧
查看 `IvyLeague/` 中的 Analysis 部分，学习招生官如何评价范文
